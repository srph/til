---
title: `sleep` suffix
---

Besides the default `seconds`, the `sleep` command accepts a suffix, like in the given example:

```bash
$ sleep 300

# with suffixes
$ sleep 5m
$ sleep 5h

# `sleep 300s` and `sleep 300` is the same
# because the default suffix is `s` / seconds.
$ sleep 300s
```
